# Analisis-Konvergensi-Gauss-Seidel-pada-SPL-Linear
Dalam project ini dilakukan proses penyelesaian SPL dengan metode Gauss-Seidel, yaitu dengan memperbarui nilai variabel secara langsung pada setiap iterasi menggunakan nilai terbaru hingga mencapai konvergensi dengan error yang semakin kecil. Perhitungan dilakukan menggunakan Microsoft Excel
